Here’s a concise prompt for formatting content in Markdown for MkDocs:

- **Headers**: Use `#` for the main title, `##` for sections, and `###` for subsections.
- **Details Box**: Use `/// details | [Title] {type: [note|info|warning|success], open: [true|false]}` to create a details box with the specified type and state.
- **Bullet Points**: Use `-` for top-level bullet points, indent with spaces for nested bullets.
- **Nested Bullets**: Use `-` for top-level bullets and indent by two spaces for each subsequent level.
- **Code Blocks**: Use triple backticks ` ``` ` for code blocks, and indent properly if nested within bullets.
- **Links and References**: Use `[Link Text](URL)` for linking to external or internal resources.
