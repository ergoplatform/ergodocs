## Safety Basics

- Always verify addresses before sending.
- Keep seed phrases offline and private.
- Treat dApp signing prompts as transactions, not popups to click through.
- Start with small test transactions when using a new app.
- If something fails, check [Access Issues](tutorials/access-issues.md), [FAQ](faq.md), or [Community](engage-with-the-ergo-community.md).
