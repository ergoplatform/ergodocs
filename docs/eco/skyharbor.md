# SkyHarbor

[skyharbor.io](https://www.skyharbor.io/) still resolves, but SkyHarbor is treated as an inactive/legacy marketplace in the ecosystem directory. Its goal was to upgrade the ENFT space with verified collections, NFT trading histories, and set-price trading.

## Project Profile

| Field | Details |
| --- | --- |
| Status | Inactive/legacy marketplace. |
| Category | NFT marketplace. |
| Links | [Website](https://www.skyharbor.io/), [frontend repository](https://github.com/skyharbor-market/frontend-skyharbor-ts). |
| Source | Public TypeScript frontend repository and marketplace site. |
| Caveats | Marketplace features, verified collection status, and API availability should be checked against the live app. |

Their roadmap includes expanded Ergo token support, NFT mass minting, NFT drops, publicly available API data, and much more!

SkyHarbor's TypeScript frontend is public in [skyharbor-market/frontend-skyharbor-ts](https://github.com/skyharbor-market/frontend-skyharbor-ts), a Next.js codebase for the marketplace UI.
