# Crux Finance

## Overview

[cruxfinance.io](https://cruxfinance.io/) will offer an all-in-one platform for portfolio management and charting within the Ergo ecosystem.

## Recent updates

- `Jan 7`: [USE analytics](http://cruxfinance.io/use-analytics) went live.
- `Dec 2025` to `Jan 2026`: Crux frontend work added trending, gainers and losers boxes, loading states, logo fixes, and USE analytics.
- `Jan 7` to `Jan 14`: Crux tracked `USE/USDT` liquidity on Binance Chain and followed the USE and DexyGold DefiLlama adapter work.
