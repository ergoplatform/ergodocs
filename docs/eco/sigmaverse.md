# Sigmaverse

## Overview

Sigmaverse is the Ergo ecosystem directory for dApps, wallets, privacy tools, infrastructure, and community projects.

## Recent updates

- `Feb 12`: backend access was restored and new projects were being added, with planned GitHub data ingestion and a GitHub category.
- `Feb 18`: dApps, privacy, explore, and wallet categories were mostly up to date, and some inactive projects were archived.
- `Mar 11`: more projects were added, UI fixes landed, unused code was removed, and the add-dApp flow moved toward a GitHub issue template.
- `Mar 18`: [PR #209](https://github.com/ergoplatform/sigmaverse/pull/209) opened to redirect "Add Your DApp" to a GitHub issue template and remove unused code.
