## Before Asking For Help

- Note the exact wallet, node, SDK, or app version.
- Save the exact error message.
- Record whether you are on mainnet, testnet, or devnet.
- Avoid sharing seed phrases, private keys, API keys, or wallet files.
- For transaction issues, share transaction IDs or public addresses only if you understand the privacy trade-off.
