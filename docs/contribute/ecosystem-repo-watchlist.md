# Ecosystem Repo Watchlist

This page gives Source Watch broad coverage for ecosystem repositories that are useful to track, but do not always justify a dedicated documentation page or narrow per-page watch metadata.

Dedicated docs pages should still use their own `source_repos` entries with narrower paths. This page is a backstop for long-tail ecosystem repos: README updates, releases, and future source signals can surface for maintainer review without forcing every project into public-facing docs.

## Scope

- Watches repository-level project signals through each repo's default branch.
- Uses `README.md` for most repos, with a few specific files or example directories where the source list pointed directly to them.
- Keeps org-only and non-GitHub project links out of `source_repos`; Source Watch needs concrete `owner/repo` targets.
